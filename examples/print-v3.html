Setting the Print V3

This example shows how we can used the MapFish Print v3

This example shows how to set the bounds of a printable PDF interactively with a dynamic print extent rectangle. It requires the MapFish Print v3.

Drag one of the handles to control the scale. Grab one of the corner handles at its edge to rotate the extent. Hold the SHIFT key to rotate in 45Â° increments only. Drag the extent rectangle to change the center.
